# Framework Comparison

## Streamlit
**Port:** 8501

**Strengths:**
- Fastest development time
- Automatic UI generation
- Built-in caching
- Great for data science workflows

**Weaknesses:**
- Limited layout control
- Full page reloads on interaction
- Less suitable for complex apps
- Harder to customize styling

**Docs:** https://docs.streamlit.io

---

## Gradio
**Port:** 7860

**Strengths:**
- Best for ML/AI demos
- Built-in chat interface
- Easy file upload handling
- Tab-based layouts
- Fast prototyping

**Weaknesses:**
- Less control over layout
- Limited component library
- Not ideal for business dashboards
- Fewer customization options

**Docs:** https://www.gradio.app/docs

---

## Dash
**Port:** 8050

**Strengths:**
- Enterprise-grade features
- Full control over layout
- Production-ready
- Advanced DataTable component
- Multi-page apps
- Most customizable

**Weaknesses:**
- Steeper learning curve
- More verbose code
- Slower development
- Callback complexity for large apps

**Docs:** https://dash.plotly.com

---
